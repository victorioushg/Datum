From the Artist

July 29th, 2003

Clipper System 3: More Apps by Gedeon Maheux

The Clipper System evolved from a set of internal icons I developed for use on The Iconfactory’s own administration web pages. I thought they turned out well enough to expand the collection to a full-blown CandyBar collection, and the 80+ icons you see here are the result.

I’ve done my best to keep the set simple in style, yet interesting and bold enough so that they work well in the toolbars and Finder windows of OS X. All of the included folders contain open/drop states, courtesy of our own IconBuilder Pro filter for Adobe Photoshop. I’ve also tried to include alternate icons of some key concepts in case you don’t like to use badged folders. I think this set will appeal to users both young and old. The simple act of dropping a file onto one of these folders still tickles me. Mindy and I thought this would be a perfect set to add GUI sounds to if we could. We can hear the folders making a “bloop bloop” sound when they change states :-)

Thanks to my loving wife, Mindy, for her patience and support over the 6+ months it took to create this collection. Love you kiddo!
